# SQL Assignment 6

## 📌 Overview
This project demonstrates SQL concepts including aggregation, filtering, sorting, and conditional classification.

---

## 🗂️ Table Structure
- OrderID
- CustomerName
- ProductCategory
- Quantity
- TotalPrice
- OrderDate

---

## ⚙️ How to Run

1. Create the Orders table 
2. Insert sample data 
3. Run all SQL queries 
4. View outputs

- Filtering (WHERE, HAVING)
- Sorting (ORDER BY)
- Conditional Logic (CASE)

## 🎯 Purpose

This assignment helps in understanding how SQL is used in real business scenarios like:
- Sales analysis

- Customer segmentation
